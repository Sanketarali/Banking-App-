Simple Banking System

This Django-based banking application offers a secure and efficient platform for managing financial transactions. It includes user authentication to ensure only authorized access. Users can perform deposits and withdrawals, with functionalities like updating account balances and validating transaction limits. The app features a transaction reporting system, allowing users to filter and view their transaction history by date range. A scheduled Celery task calculates and applies interest to eligible accounts, updating balances and recording transactions.
